PcdxParameterEncryptionZendCryptBundle
======================================

This bundle is an add-on for the `PcdxParameterEncryptionBundle`_
and enables simple usage of it in combination with the Composer package
`zendframework/zend-crypt`_.

*   :doc:`/getting-started`
